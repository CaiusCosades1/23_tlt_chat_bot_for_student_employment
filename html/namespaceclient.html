def client.run_client ( )

Метод запуска бота.

Отсюда вызывается методы:

  • редактирования временногй конфигурации (redact_config)
  • проверки подключения к базе данных (db_connection_test)
  • сохранения конфигурации (save_config) Помимо этого здесь создается обьект ядра чат-бота с заданными параметрами и запускается в прослушивание
